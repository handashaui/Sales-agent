# Sales Agent Harness

Minimal sales Agent Harness for lead follow-up, tool execution, state management,
trajectory logging, and automated evaluation.

This branch is designed for the coding exercise described in the prompt. It uses
Python and a DeerFlow-style agent harness: a clear system prompt, structured
state, mock tools, deterministic fallback behavior, and optional integrations
for real LLM calls and Langfuse scoring.

Full setup, run commands, eval instructions, architecture, and known issues are
documented after implementation.
